== Description ==
The "Animated Login Page With IP Lockdown Security" plugin allows you to enhance the login page of your WordPress website with animated elements and provides robust security features to protect against brute force login attacks.

With this plugin, you can create an engaging and visually appealing login page by customizing the appearance and behavior of the login, registration, and lost password forms. Customize the styles, add interactive elements, and even include animations using custom CSS and JavaScript files.

One of the key features of this plugin is the IP lockdown security. It protects your WordPress blog from brute force login attacks by implementing a lockdown mechanism. Whenever an excessive number of login attempts are detected from the same IP address, the plugin automatically locks down that IP address for a specified period of time. This helps prevent unauthorized access and enhances the overall security of your website.

== Features ==
- Customizes the WordPress login page with animated elements.
- Adds custom HTML and labels to the login, registration, and lost password forms.
- Allows customization of the login page's appearance using custom CSS styles.
- Enhances functionality with custom JavaScript files for interactivity and animations.
- Implements IP lockdown security to protect against brute force login attacks.
- Automatically detects excessive login attempts from the same IP address and applies temporary lockdown.
- Provides a secure and reliable login experience for your website's users.

== Installation ==
1. Upload the "animated-login-page-with-ip-lockdown-security" folder to the "/wp-content/plugins/" directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Customize the login page appearance by modifying the provided CSS styles.
4. Optionally, adjust the IP lockdown settings to meet your specific security requirements.
5. Enjoy the animated login page and enhanced security provided by the plugin!

== Frequently Asked Questions ==
1. How does the IP lockdown security feature work?
   The plugin keeps track of login attempts from each IP address. When a certain threshold of failed login attempts is reached within a defined timeframe, the IP address is temporarily locked down for a specified period. During this lockdown period, login attempts from that IP address are blocked.

2. Can I customize the login page appearance?
   Yes, you can customize the login page appearance by modifying the provided CSS styles. The plugin allows you to add your own custom styles to achieve the desired look and feel.

3. Can I add animations or interactive elements to the login page?
   Absolutely! The plugin includes support for custom JavaScript files, which you can use to add animations, interactive elements, or any other desired behavior to the login page.
